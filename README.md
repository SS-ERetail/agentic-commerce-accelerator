# 🚀 agentic-commerce-accelerator (MVP)

## 📌 Strategic Overview
As commerce shifts from "Search" to "Action," products must be discoverable not just by humans, but by **Autonomous AI Agents**. This prototype serves as a Proof of Concept (PoC) for the **Wesfarmers OneDigital Ecosystem** to optimize SKU performance in an Agentic Commerce world.

This tool identifies underperforming SKUs and leverages **Generative AI** to transform unstructured retail data into "Agent-Ready" structured content, ensuring Bunnings and Kmart products are the first choice for AI shopping assistants.

---

## 🛠️ The Three-Pillar Approach

### 1. Performance-Driven Audit (The 'Why')
Using a Streamlit-based dashboard, the MVP audits product conversion rates. 
* **PM Utility:** Allows Product Managers to set custom performance thresholds to identify "at-risk" products that are failing to convert in traditional digital channels.

### 2. Agentic SEO Engine (The 'How')
Traditional SEO focuses on keywords for humans. **Agentic SEO** focuses on structured attributes and compatibility for LLMs.
* **Mechanism:** Generates specialized System Prompts that guide LLMs to rewrite descriptions into machine-readable formats, focusing on technical specifications and "Actionable Attributes" (e.g., Model Context Protocol mapping).

### 3. RAG Pipeline Ingestion (The 'Scale')
The prototype simulates a **Retrieval-Augmented Generation (RAG)** ingestion flow.
* **MVP Feature:** It accepts raw HTML (simulating a web scrape of a Bunnings product page) and validates it against Wesfarmers ecosystem markers, preparing unstructured data for a vector database.
